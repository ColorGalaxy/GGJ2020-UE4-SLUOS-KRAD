## 游戏简介：<br>
`《SLUOS KRAD》`是一款逆向解谜游戏，你将扮演一个类似密室逃脱的工作人员，将游戏世界中玩家们通关而搞的一团糟的场景还原成未解谜的模样。<br>
主要包含：
* 悬停物体轮廓高亮(重做)
* 交互系统
* 对话系统
* 背包系统
* 粒子
* 动画
* 音效
* AI
* LowPolygon模型
* 搭建的唯美卡通场景<br>
